ModelPart

ModelPart:

All methods of ActorRenderer.ModelPart build in such a way, that you can create full render in one chain of calls.

Learn how to use
new ActorRenderer()
    .addPart("child", "parent")
        .addBox(-4, -4, -4, 4, 0, 4)
    .addPart("grandChild", "child")
        .addBox(-4, 0, -4, 4, 4, 4)
        .setOffset(0, 4, 0)
    .endPart();
