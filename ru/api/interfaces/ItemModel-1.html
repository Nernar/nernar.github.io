Interface ItemModel

Class representing item model in player's hand and/or inventory. To get an instance of this class from your code, use getFor static function.

Remarks

The coordinates of the full block in player's hand or inventory is (0, 0, 0), (1, 1, 1), so it is generally recommended to use the models that fit that bound at least for the inventory.

Since

2.0.2b20
