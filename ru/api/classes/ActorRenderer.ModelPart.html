Class ModelPart

All methods of ModelPart build in such a way, that you can create full render in one chain of calls.

Example

new ActorRenderer()
    .addPart("child", "parent")
        .addBox(-4, -4, -4, 4, 0, 4)
    .addPart("grandChild", "child")
        .addBox(-4, 0, -4, 4, 4, 4)
        .setOffset(0, 4, 0)
    .endPart();
